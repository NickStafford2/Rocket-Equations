# Earth Reference Context

Source article:
`https://sangillee.com/2024-06-07-create-realistic-earth-with-shaders/`

Purpose:
- This folder is a local reference copy of the shader structure shown in the article.
- The filenames in this folder are aligned to the article's conceptual parts:
  - `earth-vertex-shader.glsl`
  - `earth-fragment-shader.glsl`
  - `add-on-vertex-shader.glsl`
  - `atmosphere-fragment-shader.glsl`
  - `fresnel-fragment-shader.glsl`
  - `main.js`

Notes:
- `main.js` still contains placeholder import paths like `PATH_TO_EARTH_VERTEX`.
- The two fragment shaders added here were copied from the article snippets so future work has the missing atmosphere/fresnel context locally.
- This folder is reference material, not production scene code.
